Frequently Asked Questions

Is Secret Revealer - social advanced search engine free?

Secret Revealer - social advanced search engine has been designed as a free tool that aims to create awareness in society about the erosion of privacy. From the beginning I intended the tool to be used at no cost. If you want to support the development of this Facebook tool, please donate ❤ !

Is this extension affiliated with Facebook?

No, the extension does not affiliated, associated, authorized, endorsed by, or in any way officially connected with Facebook.. It's made from user for users.

What permissions does the extension require?

The extension doesn't need any permissions.

Will you store my data somewhere?

No. Your data are stored locally on your own computer. The extension does not send any data.

Are you using Facebooks Graph API?

The extension doesn't use Facebooks API. It looks at the rendered front-end (the DOM) of the Facebook page in your browser the HTML elements and their content and with JavaScript it can detect the user profile number and other data.

Will you expand to other browsers?

I believe that user awareness is very important so I hope to have the bandwidth to expand to more browsers. Currently the extension is also available for Google Chrome. If you want to support the development of this Facebook tool, please donate ❤ !
