Gixen eBay Sniper Autosnipe Settings

Thank you for using Gixen eBay Sniper Autosnipe
We appreciate you have installed this extension.

On this page you can configure the settings for the extension
Usename : username you use to sign into gixen
Password : the password you use to sign in to gixen
Bid : the default amount shown in the bid box
Main server : the default time for gixen to place the bid before the auction ends
Offset Mirror server : the mirror offset for gixen to place the bid before the auction ends
