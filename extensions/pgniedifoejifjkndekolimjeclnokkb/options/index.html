Allows you to disable or exclusively enable the rendering of certain emoticons.
Emote Set: The set of emoticons the given filter rule corresponds to, whether being Twitch.tv emotes, BetterTTV emotes, or FrankerFaceZ emotes.
Rule Type: Channel rules apply to all emotes belonging to the given channel. Emote rules apply only to the emote with the given name.
Value: Either the name of the channel or the name of the emote to filter, depending on the selected rule type.
Blacklist mode: The emotes listed in the table directly below will not be rendered.
Whitelist mode: ONLY the emotes listed in the table below will be rendered.
Allows you to disable or exclusively enable Global Twitch Emotes from running on certain websites. Asterisk wildcards supported.
Example:
"youtube.com" will trigger ONLY on the homepage of YouTube, while
"youtube.com/*" will trigger on ALL addresses starting with youtube.com, such as the following:
https://www.youtube.com
https://www.youtube.com/feed/subscriptions
https://www.youtube.com/watch?v=IW--ZGJd5Fg
