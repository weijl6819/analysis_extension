It is now possible to download the cookies for a single domain and/or all the cookies stored in Chrome.

The popup shows the cookies for current domain, and provides two different links to download both current and all cookies. e.g.:

You can navigate links as usual, e.g. on Mac OS X you can download the cookies for current domain pressing ENTER, as soon as the popup opens; and if you want to dowload all the cookies, TAB + ENTER will do the job.
